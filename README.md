# Screen Time Dashboard - Vercel Deployment

This is the Screen Time Journey dashboard deployed on Vercel.

## Features
- React-based milestone tracking dashboard
- Mobile-friendly responsive design
- Real-time username validation
- Progress tracking with levels and achievements

## Deployment

This app is configured for Vercel deployment with:
- Automatic builds from React scripts
- SPA routing support
- Optimized static file serving

## Development

To run locally:
```bash
npm install
npm start
```

To build for production:
```bash
npm run build
```

## Original Source
This app was migrated from AWS Amplify to Vercel for better performance and easier deployment management.
